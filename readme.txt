Color Mixer is a web-based application that can be used to mix the three naturally found colors to produce a tertiary color.

How To Use:

1. Input the three color amounts in numbers
2. Press the mix button
The outcome should be a mixed tertiary color on the next page. 

'About' page contains how to use it in a bit more details.
'Contact' page is about the developer.
Language: PHP
